# Пользовательское соглашение brokerapp.ru

**ООО «КонВиста Консалтинг»** (далее — *Brokerapp*) предлагает пользователю сети Интернет (далее — *Пользователь*) использовать функции портала **brokerapp.ru** (далее — *Сервис*) на условиях, изложенных в настоящем Пользовательском соглашении (далее — *Соглашение*). :contentReference[oaicite:1]{index=1}

Соглашение может быть изменено Brokerapp без какого-либо специального уведомления, новая редакция Соглашения вступает в силу с момента ее размещения в сети Интернет по указанному в настоящем абзаце адресу. Действующая редакция Соглашения всегда находится на странице по адресу: `https://brokerapp.ru/legal/rules`. :contentReference[oaicite:2]{index=2}

Начиная использовать какой-либо сервис/его отдельные функции, либо пройдя процедуру регистрации, Пользователь считается принявшим условия Соглашения в полном объеме, без всяких оговорок и исключений. В случае несогласия Пользователя с какими-либо из положений Соглашения Пользователь не вправе использовать сервисы Brokerapp. :contentReference[oaicite:3]{index=3}

Сервис предоставляется только лицам, достигшим **18 лет**. :contentReference[oaicite:4]{index=4}
