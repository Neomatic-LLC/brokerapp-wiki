# 🚗 Виджет ОСАГО - Инструкция по подключению виджета на свой сайт
## 1. Что вы получите

После сохранения виджета в приложении Brokerapp вам будет передана ссылка на **JavaScript-файл** (например, `https://brokerapp.ru/widget/widget-{unique-id}.js`).  
Когда вы вставите его на свой сайт — на странице автоматически появится готовый к работе виджет. Ничего сложного — просто копируете и вставляете.

## 2. Подготовка

### 2.1. Требования к сайту

- Поддержка JavaScript (включена по умолчанию во всех современных браузерах)
- Наличие доступа к HTML-коду страниц (или возможность установки сторонних скриптов)
- Рекомендуется: загрузка виджета через **HTTPS** (если ваш сайт работает по HTTPS, виджет тоже должен загружаться по HTTPS)

### 2.2. Что нужно знать

- Где на вашем сайте вы хотите разместить виджет
- CSS-селектор элемента, внутрь которого будет встроен виджет

## 3. Установка виджета

Вам будет предоставлена ссылка, которую надо будет прописать следующим образом:

```html

<script src="https://brokerapp.ru/widget/widget-{unique-id}.js"></script>


```


где `https://brokerapp.ru/widget/widget-{unique-id}.js` - полученная Вами ссылка на js - файл.

Всё, что нужно — скопировать этот код и вставить его на свой сайт.

При необходимости можно переопределить стиль виджета по желанию. Например, так можно добавить красно-белую тему:

```html

<script data-osago-theme="redwhite" 
src="https://brokerapp.ru/widget/widget-{unique-id}.js"></script>

```

## 4. Проверка результата

- Откройте страницу Вашего сайта в браузере
- Обновите страницу (Ctrl+F5), чтобы сбросить кеш
- Визуально оцените, появился ли виджет в нужном месте
- Откройте консоль браузера (F12 → Console) — там не должно быть красных ошибок, связанных с загрузкой виджета
